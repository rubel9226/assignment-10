# 🌱 Digital Life Lessons

A full-stack web application where users can create, organize, and share meaningful life lessons with the community. Users can save favorite lessons, react, comment, and unlock premium content through Stripe payment integration.

- 🔐 Better Auth Authentication (Email/Password & Google Login)
- 👤 User & Admin Dashboard
- 📖 Create, Update & Delete Life Lessons
